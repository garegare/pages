    ブログに RSS フィードを追加した

    2026年03月15日
    • lume
    • rss
    • tips

    やったこと

    このブログに RSS フィードを追加した。

    • /feed.xml — RSS 2.0 形式
    • /feed.json — JSON Feed 形式

    あわせてサイドバーに RSS へのリンクと GitHub リポジトリへのリンクを設置した。

    Lume の feed プラグイン

    このブログは静的サイトジェネレーターの Lume を使っている。Lume には公式の feed プラグインがあり、設定を追加するだけでフィードを生成できる。

    _config.ts に以下を追加した。

    import feed from "lume/plugins/feed.ts";

site.use(feed({
  output: ["/feed.xml", "/feed.json"],
  query: "type=post",
  sort: "date=desc",
  info: {
    title: "=site.blog_title",
    description: "=site.blog_description",
    lang: "ja",
  },
  items: {
    title: "=title",
    description: "=description",
  },
}));

    output に複数のパスを指定することで、拡張子に応じて RSS 2.0 と JSON Feed の両方を同時に生成できる。query でフィードに含めるページを絞り込んでいる。

    ブラウザ自動検出

    RSS 対応ブラウザやリーダーがフィードを自動検出できるよう、<head> に <link rel="alternate"> を追加した。

    <link rel="alternate" type="application/rss+xml" title="{{ site_name }}" href="/feed.xml">

    これにより、ブラウザの拡張機能や一部のリーダーアプリが自動的にフィードを見つけられるようになる。

    サイドバーへのリンク設置

    自動検出だけでは気づきにくいので、サイドバーにもリンクを設置した。プロフィールエリアと検索・タグエリアの間に新しいセクションを追加している。

    <section class="sidebar-section">
  <ul class="sidebar-tag-list">
    <li><a href="/feed.xml">RSS</a></li>
    <li><a href="https://github.com/garegare?tab=repositories" target="_blank" rel="noopener">GitHub</a></li>
  </ul>
</section>

    ついでに GitHub のリポジトリ一覧へのリンクも並べて置いた。
